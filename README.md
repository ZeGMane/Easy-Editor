# Приложение о редактирование картинок.

## Функционал

1. Повернуть картинку налево и направо,
2. Увеличить резкость,
3. Сделать чёрно-белым, 
4. Отзеркалить изображение, 
5. Сохранить картинку,
6. Сбросить фильры.

## Тестирование

Создал mind map и протестировал 3 сценария:

1. Резкость,
2. Отзеркаливание,
3. Сброс фильтров.
